O M3U8Player é uma extensão do navegador Chrome que implementa clientes de streaming HTTP em tempo real. Você pode reproduzir transmissões HTTP Live Streaming (HLS) no formato .m3u8. Ele depende do HTML5 para estender vídeo e fontes de mídia para reprodução. Seu princípio de funcionamento é converter o fluxo de transporte MPEG-2 e o fluxo AAC/MP3 em segmentos ISO BMFF (MP4). Se houver um Web Worker no navegador, ele pode ser usado para transcodificar de forma assíncrona. Também suporta HLS + FMP4.

O que é um arquivo m3u8?

A extensão M3U8 é um arquivo de lista de reprodução de áudio codificado em UTF-8. São arquivos de texto simples que podem ser usados tanto por reprodutores de áudio quanto de vídeo para descrever a localização de arquivos de mídia.

Por exemplo, um arquivo M3U8 pode fornecer uma referência a arquivos online para rádios da internet. Outro arquivo pode ser criado em seu computador para criar uma lista para sua música pessoal ou uma série de vídeos.

De qualquer forma, o efeito é o mesmo: você pode abrir o arquivo e começar a reproduzir rapidamente o conteúdo apontado pela lista de reprodução. Se você gosta de ouvir a mesma música várias vezes, pode criar um arquivo M3U8 como atalho para tocar essas faixas específicas em um player de mídia.

Os arquivos podem usar caminhos absolutos, caminhos relativos e URLs para apontar para arquivos de mídia específicos e/ou para uma pasta inteira de arquivos de mídia. Outras informações no arquivo podem ser comentários descrevendo o conteúdo.

M3U8 é um formato de arquivo para listas de reprodução de multimídia, originalmente projetado para arquivos de áudio, como MP3. No entanto, cada vez mais softwares também o utilizam para listas de vídeos, e o M3U8 pode especificar fontes de streaming de áudio online. Vários dispositivos e programas suportam o formato M3U8.

Como abrir um arquivo m3u8?

A maioria dos editores de texto pode editar e ler arquivos M3U8, incluindo o Bloco de Notas do Windows. No entanto, abrir o arquivo com o Bloco de Notas permite apenas ler as referências dos arquivos, não reproduzi-los, pois editores de texto são diferentes de reprodutores ou gerenciadores de mídia.

Se você está procurando um bom programa para abrir e usar arquivos M3U8, pode experimentar o VLC, iTunes ou Songbird. Outra maneira de abrir arquivos M3U8 no Linux é usar o XMMS, enquanto o CocoModX é indicado para usuários de Mac (além de alguns programas compatíveis com Windows).

Você pode usar o reprodutor m3u8 online para abrir arquivos rapidamente. No entanto, se o M3U8 estiver armazenado em seu computador ou dispositivo, o site pode não funcionar, pois é necessário ter a URL do arquivo e o conteúdo referenciado deve estar online.
