# 快速开始 - M3U8播放器部署

## 1. 确认位置
确保你在正确的项目目录：
```powershell
cd "D:\桌面\m3u8player\m3u8播放器"
```

## 2. 一键部署
```powershell
# PowerShell 执行
.\deploy.ps1 "初始化静态工具站"

# 或使用 Python
python push_to_github.py "初始化静态工具站"

# 或手动执行
git add -A; git commit -m "初始化静态工具站"; git push origin main
```

## 3. 等待部署
推送到 GitHub 后，Vercel 会自动部署（约1-2分钟）。

## 4. 验证站点
访问以下链接确认部署成功：
- 首页: https://m3u8player.gitlcp.com/
- ads.txt: https://m3u8player.gitlcp.com/ads.txt
- sitemap.xml: https://m3u8player.gitlcp.com/sitemap.xml

## 5. AdSense设置
1. 登录 https://www.google.com/adsense
2. 添加站点 → 输入 `m3u8player.gitlcp.com`
3. 选择 "我已放置 ads.txt"
4. 等待审核（1-2周）
